# DataFusionSharp Package Smoke Test

## Purpose

This is a simple smoke test project to verify that the DataFusionSharp package works correctly after installation. It performs a basic sanity check to ensure:

- The package can be referenced and used in a .NET project
- The runtime initializes properly
- Basic SQL queries execute successfully
- Simple results are returned without errors

## Usage

This project is intended to be run as part of a CI/CD pipeline or as a manual test after installing the DataFusionSharp package.

Check CI/CD configuration for details on how to execute the tests.
